Scholastic project involving the implementation of a platoon maneuver using Omnet++, Sumo, Veins and Plexe.

The maneuver consist in the cooperation between a platoon and a independent vehicle, in case of emergency they need to cooperate to let the overtaker return to the mainlane in safety.

The code is a clone of Plexe from Michele Segata with the implementation of the new maneuver.
